# Sistem Manajemen Playlist Musik

Program Java Playlist Musik merupakan aplikasi sederhana berbasis bahasa pemrograman Java yang digunakan untuk mensimulasikan pengelolaan daftar lagu (playlist). Melalui program ini, pengguna dapat menambahkan lagu, melihat daftar lagu, serta mengelola playlist secara sederhana melalui terminal atau console.

## Deskripsi Program

Program ini adalah implementasi praktis dari operasi array dasar dalam bahasa Java. User dapat mengelola playlist musik dengan fitur-fitur berikut:

### Fitur Utama:
1. Menampilkan semua lagu di playlist
2. Menambahkan lagu baru (dengan pengecekan kapasitas)
3. Menghapus lagu berdasarkan judul (dengan penggeseran elemen)
4. Mencari lagu menggunakan Linear Search
5. Mengurutkan lagu berdasarkan durasi (Selection Sort, ascending)

## Cara Menjalankan

### 1. Clone Repository
```bash
git clone https://github.com/ridwansyahlatjeno-lab/Playlist.git
cd Playlist
```

### 2. Compile Program
```bash
javac *.java
```

### 3. Jalankan Program
```bash
java PlaylistArray
```
