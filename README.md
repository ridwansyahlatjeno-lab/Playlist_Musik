# Java Playlist Lagu

## Deskripsi Program
Program **Java Playlist Lagu** merupakan aplikasi sederhana berbasis bahasa pemrograman Java yang digunakan untuk mensimulasikan pengelolaan daftar lagu (playlist). Melalui program ini, pengguna dapat menambahkan lagu, melihat daftar lagu, serta mengelola playlist secara sederhana melalui terminal atau console.

Program ini dibuat sebagai bagian dari latihan memahami konsep dasar **Object Oriented Programming (OOP)** dalam Java seperti penggunaan class, object, method, dan pengelolaan data menggunakan struktur list.


## Cara Menjalankan Program

Ikuti langkah-langkah berikut untuk menjalankan program di komputer lokal.

### 1. Clone Repository

Clone repository ini menggunakan perintah berikut:

```bash
git clone https://github.com/defghizz/Java-PlaylistLagu.git
```

### 2. Compile

Untuk compile menggunakan perintah berikut:

```bash
javac *.java
```

### 3. Jalankan

Jalankan Program dengan perintah berikut:

```bash
java PlaylistOOP
```
